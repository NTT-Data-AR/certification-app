# Metricas KPI

| KPI | Definicion |
| --- | --- |
| Cobertura obligatoria | Cumplidas/requeridas |
| Brecha | Requeridas no cubiertas |
| Vencimientos | Expiran en ventana |
| Tiempo validacion | Dias envio-decision |
| Tasa rechazo | Rechazadas/validadas |
| Tool success | Tools exitosas/totales |
| Denegaciones correctas | Bloqueos esperados |
