# Casos de uso

## CU-001 Catalogo

Owner busca duplicados, crea o actualiza certificacion, define vigencia/evidencia y audita.

## CU-002 Asignacion

Manager selecciona colaborador, certificacion activa, prioridad, fecha objetivo y confirma.

## CU-003 Registro

Colaborador informa emision, expiracion, credential id y evidencia; queda pendiente de validacion.

## CU-004 Validacion

Validador revisa evidencia y aprueba, rechaza o pide correccion con motivo.

## CU-005 Consulta IA

Usuario pregunta; agente verifica permisos, invoca tool y responde con datos autorizados.

## CU-006 Escritura IA

Agente prepara resumen, pide confirmacion, invoca tool y audita.
