# Etapa Power Apps

## Rol

La etapa Power Apps sirve para recuperar formularios, columnas, flujos, reglas y adopcion de usuarios.

## Riesgos

| Dimension | Riesgo |
| --- | --- |
| Modelo | Listas orientadas a pantalla, no normalizadas |
| Gobierno | Dependencia de makers y ambientes |
| ALM | Versionado incompleto |
| Seguridad | Permisos heredados de conectores |
| Performance | Limites de delegacion o consultas |

## Migracion

Extraer metadata, perfilar datos, mapear a modelo canonico, validar con negocio y cargar mediante lote auditado.
