# Privacidad de datos

## Datos personales

Nombre, email, employee number, unidad, manager, pais, certificaciones personales y evidencias.

## Principios

Licitud, transparencia, limitacion de finalidad, minimizacion, exactitud, retencion limitada, integridad/confidencialidad y responsabilidad.

## Agente IA

No mostrar terceros sin permiso, no retener evidencias, minimizar respuestas y rechazar solicitudes fuera de alcance.
