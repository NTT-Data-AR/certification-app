# Gobierno de datos

## Roles

| Rol | Responsabilidad |
| --- | --- |
| Data Owner | Define reglas y acceso |
| Data Steward | Calidad y linaje |
| Security Owner | Controles |
| Product Owner | Prioridad |
| AI Owner | Uso de datos por IA |

## Clasificacion

| Nivel | Ejemplos |
| --- | --- |
| Publico | Nombres publicos de certificaciones |
| Interno | Catalogos y estados |
| Personal | Email, employee number, certificaciones |
| Confidencial | Evidencias y auditoria detallada |
