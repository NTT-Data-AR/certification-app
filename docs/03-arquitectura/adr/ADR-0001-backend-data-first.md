# Backend como fuente de verdad

## Estado

Propuesta

## Contexto

La evolucion hacia IA requiere control, trazabilidad y separacion entre experiencia conversacional y datos.

## Decision

El backend canonico sera la unica fuente de verdad; el agente solo usa APIs/tools.

## Consecuencias

Mayor control y seguridad; requiere contratos y disciplina de versionado.
