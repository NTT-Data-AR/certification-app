# Alcance funcional

## Incluido

Catalogo, colaboradores, requerimientos, asignaciones, registros, evidencias, validacion, vencimientos, renovacion, reporting, auditoria y agente IA.

## Fuera de alcance

LMS completo, compras de vouchers, integraciones productivas, OCR como validacion unica, decisiones laborales automatizadas.

## Criterios de exito

Requerimientos trazables, modelo de datos validado, diccionario completo, APIs iniciales, flujos IA testeables y controles de seguridad.
