# Runbook incidentes

## Datos personales

Contener, preservar logs, identificar alcance, notificar security/privacy, revocar permisos, RCA y remediacion.

## Accion IA incorrecta

Revisar conversation/tool ids, validar backend, revertir auditadamente si aplica, actualizar guardrails y tests.

## Job fallido

Verificar ultima ejecucion, revisar error, reprocesar idempotente y validar duplicados.
