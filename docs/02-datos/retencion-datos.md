# Retencion de datos

## Criterio

Validar con politicas internas y privacidad. Evidencias y conversaciones IA requieren retencion minimizada.

## Tabla

| Dato | Retencion sugerida |
| --- | --- |
| Catalogo | Vigente + historico |
| Registros | Relacion laboral + periodo legal |
| Evidencias | Segun politica y necesidad |
| Auditoria | Periodo legal/contractual |
| Conversaciones IA | Minimizada y expirable |
| Tool invocations | Mantener para auditoria |
