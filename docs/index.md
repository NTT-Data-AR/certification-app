# Indice de documentacion

- 00-contexto: vision, antecedentes web, Power Apps, enfoque IA, fuentes y supuestos.
- 01-analisis-funcional: alcance, actores, procesos, requerimientos, reglas, casos, trazabilidad.
- 02-datos: modelo conceptual/logico, diccionario, entidades, calidad, gobierno, retencion, linaje, mercado y formacion.
- 03-arquitectura: arquitectura general, backend, agente, integraciones y ADRs.
- 04-procesos: procesos operativos documentados.
- 05-seguridad-cumplimiento: seguridad, privacidad, permisos, auditoria, cumplimiento, amenazas y riesgos IA.
- 06-agent-ai: objetivos, capacidades, restricciones, prompts, flujos, tools, guardrails y evaluacion.
- 07-testing: estrategia y casos de prueba.
- 08-operacion: monitoreo, soporte, runbooks y KPIs.
