# Flujos conversacionales

## Consulta vencimientos

Usuario pregunta, agente valida alcance, invoca `list_due_renewals`, resume riesgos y acciones.

## Registro evidencia

Agente busca certificacion, pide datos faltantes, resume, confirma e invoca `register_certification_evidence`.

## Asignacion

Agente valida manager, completa parametros, confirma e invoca `create_assignment`.
