# Capacidades

## Consulta

Buscar catalogo, ver propias, vencimientos, asignaciones, reglas y cobertura autorizada.

## Accion

Crear asignaciones, registrar evidencia/borrador, abrir incidencia y generar reporte autorizado con confirmacion.

## Excluido

Aprobar certificaciones, cambiar permisos, modificar auditoria, SQL libre o decisiones laborales.
