# Actores y stakeholders

## Actores

| Actor | Responsabilidad |
| --- | --- |
| Colaborador | Consulta y registra certificaciones propias |
| Manager | Gestiona cobertura y asignaciones de equipo |
| Certification Owner | Mantiene catalogo y reglas |
| Validador | Aprueba o rechaza evidencias |
| Data Steward | Gestiona calidad y linaje |
| Admin | Configura parametros y roles |
| Auditor | Consulta historico |
| Agente IA | Orquesta herramientas bajo permisos |

## RACI resumido

| Actividad | R | A | C/I |
| --- | --- | --- | --- |
| Registrar evidencia | Colaborador | Colaborador | Validador |
| Validar | Validador | Certification Owner | Manager |
| Mantener catalogo | Owner | Owner | Steward |
| Asignar | Manager | Manager | Owner |
| Corregir datos | Steward | Data Owner | Admin |
